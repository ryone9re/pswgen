# pswgen

A simple random password generator execute in CLI

## Installation

If you can use golang, please run:

```bash
go install
```

## Usage

```bash
pswgen [OPTIONS] [PASSWORD LENGTH]
```

OPTIONS:
    -h, --help:
        Show this help message and exit.
    -v, --version:
        Show version and exit.
    -s, --symbol:
        Add symbol to password.
